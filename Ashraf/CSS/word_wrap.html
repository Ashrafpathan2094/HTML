The Word-Wrap Property

word-wrap: normal (defualt):

This Div Contains a Very Long Word: thisisaveryveryveryveryveryveryveryveryverylongword. This long word will not break and wrap to the next line

word-wrap: break-word:

This Div Contains a Very Long Word: thisisaveryveryveryveryveryveryveryveryverylongword. This long word will not break and wrap to the next line
